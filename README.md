# NodeAWS

### Install Node js on AWS server

`sudo apt update`

Optional : `curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -`

`sudo apt install nodejs`
`sudo apt install npm`
`npm i`

### Install Node packages

`npm i express`

### Run API on AWS

`node index.js`

## Deploy React on AWS

`git checkout React`
